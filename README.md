# Student Management System 🎓

A simple, console-based Java application to manage student records efficiently. This project is built using pure Java without any heavy build tools like Maven or Gradle, making it lightweight and easy to run.

## 🚀 Features
- **Register Students:** Add new students with a unique ID, Name, and Course.
- **View Records:** Display all registered students in a clean, tabular format.
- **Delete Records:** Remove student entries using their unique ID.
- **Input Validation:** Handles basic menu navigation.

## 🛠️ Technologies Used
- **Language:** Java (JDK 17+)
- **IDE:** IntelliJ IDEA
- **Storage:** In-memory ArrayList (Data persists during the session)
